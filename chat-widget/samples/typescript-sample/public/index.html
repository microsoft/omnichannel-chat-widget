Omnichannel Chat Widget - TypeScript Sample

Configuration Required

To use this sample, you need to configure your Omnichannel settings in the getOmnichannelChatConfig function in src/index.tsx:

  • orgId: Your organization ID
  • orgUrl: Your organization URL
  • widgetId: Your widget/app ID

Replace the placeholder values like <DATA-ORG-ID> with your actual configuration values.
