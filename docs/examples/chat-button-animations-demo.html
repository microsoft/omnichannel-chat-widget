Chat Button Animation Demo

This demo shows the various animations available for the chat button component using external CSS.

1. Shake on Load Animation

Shakes 3 times with a 0.5s delay (ideal for page load)

liveChatWidgetProps = {
  ...liveChatWidgetProps,
  chatButtonProps: {
    styleProps: {
      generalStyleProps: {
        animation: "chatButtonShake 0.5s ease-in-out 3 0.5s"
      }
    }
  }
};

2. Bounce on Load Animation

Bounces 2 times with a 1s delay

liveChatWidgetProps = {
  ...liveChatWidgetProps,
  chatButtonProps: {
    styleProps: {
      generalStyleProps: {
        animation: "chatButtonBounce 0.6s ease-in-out 2 1s"
      }
    }
  }
};

3. Continuous Pulse Animation

Continuous scaling effect

liveChatWidgetProps = {
  ...liveChatWidgetProps,
  chatButtonProps: {
    styleProps: {
      generalStyleProps: {
        animation: "chatButtonPulse 1s ease-in-out infinite"
      }
    }
  }
};

4. Continuous Glow Animation

Continuous glowing shadow effect

liveChatWidgetProps = {
  ...liveChatWidgetProps,
  chatButtonProps: {
    styleProps: {
      generalStyleProps: {
        animation: "chatButtonGlow 2s ease-in-out infinite"
      }
    }
  }
};
